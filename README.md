# Week 1: Chrome Extension Research

> Исследование рынка Chrome-расширений для создания прибыльного продукта.
> Курс: **Captain Builders Bootcamp** (monetize.software)

---

## О проекте

Этот репозиторий содержит материалы и результаты первой недели буткемпа по созданию Chrome-расширений. Цель — выбрать и валидировать идею для расширения, которое можно монетизировать.

### Методология

Оценка идей по **8 критериям** (макс. 50 баллов):

| # | Критерий | Описание |
|---|----------|----------|
| 1 | Пользователи | Потенциальная аудитория (10K-1M) |
| 2 | Заработок | Возможность монетизации |
| 3 | Одна функция | Можно ли свести к 1 функции |
| 4 | Простота | Сложность разработки |
| 5 | Поиск ключа | Volume запросов в US |
| 6 | Софтовость | % софта в Google выдаче |
| 7 | Ключ свободен | Нет оптимизированных конкурентов |
| 8 | KD | Keyword Difficulty |
