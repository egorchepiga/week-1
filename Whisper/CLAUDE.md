# Whisper — Инструмент транскрибации

> **Для AI:** Это инструмент для транскрибации аудио/видео файлов с использованием OpenAI Whisper

---

## Что это

Локальный инструмент для автоматической транскрибации аудио и видео файлов в текст. Использует модель OpenAI Whisper (large-v3) с GPU-ускорением.

---

## Структура папки

```
Whisper/
├── CLAUDE.md          # Этот файл
├── transcribe.bat     # Скрипт запуска транскрибации
├── check_gpu.py       # Проверка доступности GPU
├── input/             # Входные аудио/видео файлы
│   └── *.mp4, *.mp3   # Исходные файлы для транскрибации
└── output/            # Результаты транскрибации
    └── *.txt          # Текстовые файлы с транскрипцией
```

---

## Как использовать

### 1. Подготовка
```bash
# Проверить доступность GPU
python check_gpu.py
```

### 2. Транскрибация
```bash
# Положить файл в input/
# Запустить:
transcribe.bat
```

### 3. Результат
- Текстовые файлы появятся в `output/`
- Формат: построчная транскрипция с нумерацией

## Промпты для обработки транскрипций

### Промпт 1: Извлечение ошибок
```
Прочитай транскрипцию разбора ошибок буткемпа.
Извлеки все упомянутые ошибки при выборе идей для Chrome Extension.
Для каждой ошибки укажи:
- Название ошибки
- Описание проблемы
- Почему это плохо
- Как исправить
Формат: Markdown таблица или список
```

### Промпт 2: Создание выжимки
```
Прочитай транскрипцию по частям (файл большой).
Создай структурированную выжимку:
1. Главные инсайты (3-5 пунктов)
2. Разобранные идеи с вердиктами
3. Ключевые правила
4. Лучшие ниши
5. Типичный флоу валидации
Формат: Markdown с таблицами
```

### Промпт 3: Каталог ошибок для AI
```
На основе транскрипций создай каталог ошибок для AI-агента.
Формат должен позволять:
- Быстро проверить идею на типичные ошибки
- Получить чек-лист валидации
- Понять критичность каждой ошибки
Группировать по категориям: Критические, Оценка запроса, Оптимизация, Ниша
```

## Технические детали

### transcribe.bat
```batch
whisper --model large-v3 --device cuda --language ru input/*.mp4 -o output/
```

### check_gpu.py
Проверяет доступность CUDA и GPU для ускорения транскрибации.
