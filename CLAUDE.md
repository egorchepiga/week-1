## Промпты для продолжения работы

### Анализ ниши:
```
Проанализируй нишу Chrome-расширений "[НИША]":
1. Топ-5 конкурентов и их метрики
2. Модели монетизации
3. Незанятые ключевые слова
4. Рекомендация для входа
```

### Проверка ключа:
```
Оцени ключевое слово "[КЛЮЧ]" для Chrome-расширения:
1. Объём запросов (US/мир)
2. Keyword Difficulty
3. Софтовость выдачи (% софта в Google)
4. Занятость конкурентами
5. Рекомендация: брать/не брать
```

### Генерация MVP:
```
Создай минимальное Chrome расширение (Manifest V3):
Функция: [ОПИСАНИЕ]
Требования:
- Только 1 функция
- Готово к публикации
- Структура для локализации (_locales)
```
