## Следующие шаги для валидации

### 1. Keyword Research (Semrush)
```bash
python3 scripts/keyword_analyzer.py <keyword>
```
Проверить для выбранных ниш:
- regex tester chrome extension
- bluesky extension
- ebay listing optimizer
- xpath tester
- clipboard sync extension

### 2. Competitor Deep-dive
```bash
python3 scripts/mongodb/query-extensions.py hybrid-search "<niche>" --limit 20
python3 scripts/mongodb/query-extensions.py get <extension_id>
```

### 3. MVP Validation Checklist
```
□ Keyword Volume >= 500 (US)
□ KD <= 70%
□ SERP = софт (не статьи)
□ Конкурентов 1-30
□ Платные конкуренты существуют
□ B2B потенциал
□ MV3 совместимо
□ Нет ToS конфликтов
```
