## Расчёт выручки

```python
# Месячная выручка от подписки
monthly_revenue = (users / 100) * subscription_price
# Конверсия: 1% базовая, 3-5% оптимизированная, 10% рекорд

# Пример: 10,000 юзеров × $5/мес = $500/мес
```

---

## Продажа проекта

```python
# Стоимость проекта без выручки
sale_price = users / 10

# Пример: 100,000 юзеров = $10,000
```

# Из US-запросов в глобальный
