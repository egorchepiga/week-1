# Этап 2: Оценка доходности продуктов

## Цель этапа
Рассчитать потенциальную выручку для каждой найденной идеи.

## Формулы расчёта

### Базовая выручка с продажи расширения
```
ВЫРУЧКА_ПРОДАЖА = ПОЛЬЗОВАТЕЛИ / 10

Пример: 100,000 пользователей → $10,000/мес
```

### Выручка от подписок (Premium/Pro)
```
ВЫРУЧКА_ПОДПИСКА = (ПОЛЬЗОВАТЕЛИ / 100) × ЦЕНА_ПОДПИСКИ

Где:
- Конверсия в подписку ≈ 1% (для TIER1 стран)
- Минимальная цена подписки = $3/мес

Пример: 100,000 пользователей × 1% × $3 = $3,000/мес
```

### Мультипликатор для дорогих подписок
```
ЕСЛИ цена подписки > $3:
  ВЫРУЧКА = (ПОЛЬЗОВАТЕЛИ / 100) × РЕАЛЬНАЯ_ЦЕНА

Пример: 50,000 пользователей, подписка $9.99/мес
  → 50,000 / 100 × $9.99 = $4,995/мес
```

## Алгоритм действий

### Шаг 2.1: Проверка модели монетизации конкурента
```
ДЕЙСТВИЕ: Открыть расширение конкурента и проверить:
  1. Есть ли Premium/Pro версия?
  2. Какая цена подписки?
  3. Какие функции платные?
  4. Есть ли разовые покупки?

ГДЕ ИСКАТЬ:
  - Страница расширения в CWS
  - Сайт разработчика
  - Внутри самого расширения (установить)
```

### Шаг 2.2: Расчёт потенциала для каждой идеи
```
ДЛЯ КАЖДОЙ ИДЕИ:
  1. Записать количество пользователей конкурента
  2. Определить модель монетизации
  3. Рассчитать ВЫРУЧКА_ПРОДАЖА
  4. Рассчитать ВЫРУЧКА_ПОДПИСКА (если применимо)
  5. Записать итоговую оценку в таблицу
```

### Шаг 2.3: Присвоение баллов за монетизацию
```
БАЛЛ = 1-5 в зависимости от потенциала:

5 баллов: Подписка $10+, B2B ниша
4 балла: Подписка $5-10, профессиональные инструменты
3 балла: Подписка $3-5, массовый рынок
2 балла: Только продажа расширения, нет подписки
1 балл: Сложно монетизировать
```

## Подсказки для промптов

### Для анализа монетизации:
```
Проанализируй модель монетизации расширения [НАЗВАНИЕ]:
1. Какие функции бесплатные vs платные?
2. Какая структура ценообразования?
3. Есть ли trial period?
4. Какой средний чек?
```

### Для расчёта потенциала:
```
Рассчитай потенциальную выручку для расширения в нише [НИША]:
- Конкурент имеет [X] пользователей
- Подписка стоит $[Y]/мес
- Рынок: [B2B/B2C]
Учти конверсию и региональные особенности.
```

### Для сравнения идей:
```
Сравни потенциал монетизации 3 идей:
1. [Идея 1] - [пользователи] - [модель]
2. [Идея 2] - [пользователи] - [модель]
3. [Идея 3] - [пользователи] - [модель]
Какая имеет лучший ROI?
```

## Красные флаги (низкий потенциал)

1. **Нет платных аналогов** — рынок не готов платить
2. **Слишком простая функция** — сложно обосновать подписку
3. **Много бесплатных альтернатив** — ценовая война
4. **Узкая аудитория + низкая цена** — математика не сходится

## Зелёные флаги (высокий потенциал)

1. **B2B ниша** — компании платят больше
2. **Экономия времени** — легко обосновать ценность
3. **Интеграция с платными сервисами** — аудитория уже платит
4. **Профессиональные инструменты** — высокая готовность платить
