# Промпты и алгоритмы для AI

## Содержимое папки

Алгоритмы действий для каждого этапа создания Chrome-расширения.
Извлечены из курса Captain Builders и оптимизированы для использования с Claude AI.

| Файл | Этап | Описание |
|------|------|----------|
| 00_MASTER_ALGORITHM.md | - | Общий план всех этапов |
| 01_vybor_idei.md | 1 | Выбор идеи расширения |
| 02_ocenka_dohodnosti.md | 2 | Оценка потенциальной выручки |
| 03_podschet_ballov.md | 3 | Подсчёт баллов и выбор лучшей |
| 04_05_podbor_zaprosa.md | 4-5 | Подбор ключевого слова |
| 06_keyword_difficulty.md | 6 | Анализ сложности ключа |
| 07_proverka_konkurentov.md | 7 | Проверка конкурентов |
| 08_09_ranzhirovanie_kluch.md | 8-9 | Хвосты и софтовость |
| 10_razrabotka.md | 10 | Разработка MVP |

---

## Как использовать

### Вариант 1: Последовательно по этапам
```
1. Открыть 01_vybor_idei.md
2. Выполнить алгоритм
3. Записать результаты
4. Перейти к следующему этапу
```

### Вариант 2: Загрузить в Claude
```
Вот алгоритм для этапа [N]:
[ВСТАВИТЬ СОДЕРЖИМОЕ ФАЙЛА]

Выполни этот алгоритм для:
- Ниша: [ВАША НИША]
- Данные: [ВАШИ ДАННЫЕ]
```

### Вариант 3: Использовать готовые промпты
В каждом файле есть секция "Подсказки для промптов" с готовыми запросами.

## Ключевые пороги

| Параметр | Минимум | Оптимум |
|----------|---------|---------|
| Пользователи конкурента | 10K | 100K+ |
| Объём запросов US (широкая ниша) | 2000 | 5000+ |
| Объём запросов US (узкая ниша) | 500 | 1000+ |
| Keyword Difficulty | ≤70% | ≤50% |
| Релевантных хвостов | >60% | >80% |
| Софтовой выдачи | >50% | >70% |
| Описание расширения | 3000 символов | 5000+ |
| Количество переводов | 30 | 50+ |

## Быстрый старт

Если хочешь начать прямо сейчас:

1. **Прочитай** `00_MASTER_ALGORITHM.md` для общего понимания
2. **Выбери нишу** из топ-5 рекомендаций (см. reports/ANALYSIS_SUMMARY.md)
3. **Проверь ключ** используя `04_05_podbor_zaprosa.md`
4. **Проверь конкурентов** используя `07_proverka_konkurentov.md`
5. **Создай MVP** используя `10_razrabotka.md`
