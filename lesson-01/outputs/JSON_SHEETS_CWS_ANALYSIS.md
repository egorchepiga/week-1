## Почему ниша свободна в CWS?

1. **Workspace Add-ons работают внутри Sheets** — функция `=IMPORTJSON()` удобнее
2. **Пользователи ищут в Marketplace**, а не в CWS
3. **Разные подходы:** Add-on = формула в ячейке, Extension = кнопка в браузере

**НО:** если сделать расширение с хорошим SEO под ключ "import json google sheets", можно перехватить трафик из поисковиков!

## Функционал MVP

### Вариант 1: Import JSON → Sheets
```
1. Пользователь копирует JSON в буфер
2. Открывает Google Sheets
3. Кликает на расширение
4. JSON автоматически парсится в таблицу
```

### Вариант 2: Export Sheets → JSON
```
1. Пользователь выделяет данные в Sheets
2. Кликает на расширение
3. Получает JSON в буфер или файл
```

### Вариант 3: Двусторонний конвертер
```
1. Кнопка "Import JSON" — вставка из буфера
2. Кнопка "Export JSON" — копирование в буфер
```
