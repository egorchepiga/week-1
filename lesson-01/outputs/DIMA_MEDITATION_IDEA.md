## Изюминка идеи

### Уникальное позиционирование

Не просто "ещё одно meditation app", а **инструмент для работников**:

```
"Снимай стресс за 2 минуты не уходя из браузера"
```

### Ключевые фишки

1. **Визуальная анимация дыхания**
   - Расширяющийся/сжимающийся круг
   - Синхронизация с вдохом/выдохом
   - Минималистичный дизайн

2. **Готовые техники дыхания**
   - 4-7-8 (расслабление)
   - Box Breathing (концентрация)
   - 2-минутный calm down
   - Кастомные настройки

3. **Интеграция в рабочий процесс**
   - Popup прямо в браузере
   - Опционально: New Tab page
   - Напоминания "Time for a breath break"

4. **Приятные звуки**
   - Колокольчик начала/конца
   - Ambient фон (опционально)
   - Звук тишины (для офиса)
