## Критические проблемы

### 1. Gmail имеет встроенные шаблоны

Gmail нативно поддерживает до 50 шаблонов:
- Settings → Advanced → Templates → Enable
- Compose → More options → Templates

**Влияние:** Многие пользователи не будут искать расширение.

### 2. Ключ не софтовый

В Google выдаче на первом месте — официальная справка Gmail, не расширения Chrome.

**Влияние:** SEO трафик будет значительно ниже ожидаемого.

### 3. Сильные конкуренты

- **cloudHQ** — 200K users, полностью оптимизирован под ключ
- **Text Blaze** — 700K users, универсальное решение
- **Briskine** — активно развивается, хорошие отзывы

### 4. Сложность разработки

- Требует работы с Gmail Compose API
- Необходимо хранение шаблонов (sync storage / backend)
- Интеграция с Gmail UI — нетривиальная задача
- Не "одна простая функция"
