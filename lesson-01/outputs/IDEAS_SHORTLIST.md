# Shortlist идей для проверки — Week 1

> **Дата:** 2025-12-05
> **Источник:** Анализ базы 5625 расширений из app-database.com

---

## Критерии отбора

- ✅ 10K-300K пользователей у конкурентов
- ✅ MV3 (Manifest V3)
- ✅ Низкий рейтинг конкурентов (< 4.0) = возможность улучшить
- ✅ Простая 1 функция
- ✅ Developer Tools / Productivity ниша
- ✅ Для работы (не развлечение)

---

## 10 идей для keyword research

### 1. CSS Inspector / CSS Viewer
**Конкуренты:**
- CSS Viewer for Google Chrome™ — 50K users, 4.1 rating
- Visual CSS Editor — 50K users, 3.5 rating
- CSS Scan — 10K users, 3.7 rating
- Live editor for CSS — 70K users, 4.6 rating

**Потенциальные ключи:** `css inspector`, `css viewer`, `css checker`

**Преимущества:**
- Простая функция (показать CSS элемента)
- Много open source (CSSViewer на GitHub)
- Низкие рейтинги конкурентов

---

### 2. JSON Formatter / JSON Viewer
**Конкуренты:**
- JSON Viewer — 80K users, 4.2 rating
- JSON Formatter — 40K users, 4.4 rating
- JSON Beautifier and Editor — 30K users, 4.8 rating

**Потенциальные ключи:** `json formatter`, `json viewer`, `json beautifier`

**Преимущества:**
- Очень простая функция
- Много open source решений
- Стабильный спрос у разработчиков

---

### 3. Color Picker / Eyedropper
**Конкуренты:**
- Smart Color Picker — 100K users, 4.3 rating
- Color Picker | Eyedropper — 100K users, 4.0 rating
- Hex Color Picker — 80K users, 2.9 rating (слабый!)
- Color by Fardos — 60K users, 2.8 rating (слабый!)

**Потенциальные ключи:** `color picker`, `eyedropper`, `color grabber`

**Преимущества:**
- Простейшая функция
- Есть конкуренты с рейтингом 2.8-2.9 = легко обогнать
- Большой volume (дизайнеры + разработчики)
