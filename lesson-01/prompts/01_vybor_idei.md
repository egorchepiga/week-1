# Этап 1: Выбор идеи для Chrome-расширения

## Цель этапа
Найти перспективную идею для Chrome-расширения с потенциалом монетизации.

## Алгоритм действий

### Шаг 1.1: Подготовка инструментов
```
ДЕЙСТВИЕ: Создать копию таблицы для анализа идей
ССЫЛКА: docs.google.com/spreadsheets/d/1kfQFJjAimtTAndMV2cg45FL8RV8KdYwkyVzhttdHPgw
РЕЗУЛЬТАТ: Личная копия таблицы с колонками: Название, Ссылка, Пометки, Тематика, Количество пользователей
```

### Шаг 1.2: Активация доступа к базе данных расширений
```
ДЕЙСТВИЕ: Зарегистрироваться в Telegram-боте буткемпа
ССЫЛКА: https://t.me/the_bootcamp_bot?start=ln8Z7VIqMm43LTyM-UjMo
ВАЖНО: Использовать тот же email, что и на платформе курса
ЗАТЕМ: Перейти на app-database.com/profile и войти через Google
ЗАТЕМ: Привязать Telegram для активации бесплатного доступа
```

### Шаг 1.3: Поиск идей через Chrome Web Store
```
ДЕЙСТВИЕ: Перейти на chromewebstore.google.com/category/extensions
КРИТЕРИИ ОТБОРА:
  ✓ Тематика вам понятна и нравится
  ✓ Понимаете, что расширение делает
  ✓ Количество пользователей ≥ 10,000

МЕТОДЫ ПОИСКА:
  - Просматривать разные категории
  - Использовать поиск CWS
  - Смотреть "похожие расширения" у хороших продуктов
```

### Шаг 1.4: Поиск идей через app-database.com
```
ДЕЙСТВИЕ: Использовать базу всех расширений
ФИЛЬТРЫ:
  - Сортировка по количеству оценок (не пользователей!) — находит продукты с вовлечённой аудиторией
  - Фильтр по словам в названии (name): ai, chatgpt, productivity и др.

ПРЕИМУЩЕСТВО: База парсится ежедневно, актуальные данные
```

## Критерии хорошей идеи

| Критерий | Минимум | Идеально |
|----------|---------|----------|
| Пользователей у конкурента | 10K | 50K+ |
| Понятность функционала | Да | Да |
| Возможность упростить | Да | До 1 функции |
| Потенциал монетизации | Есть | Подписка/Premium |

## Чек-лист завершения этапа

- [ ] Таблица создана и заполнена
- [ ] Найдено минимум 5 идей
- [ ] Для каждой идеи указаны: название, ссылка, пользователи, тематика
- [ ] Идеи соответствуют критериям отбора

## Подсказки для промптов

### Для анализа ниши:
```
Проанализируй нишу Chrome-расширений в категории [КАТЕГОРИЯ].
Найди топ-10 расширений по количеству пользователей.
Определи общие паттерны успешных продуктов.
```

### Для генерации идей:
```
На основе расширения [НАЗВАНИЕ] с [X] пользователями предложи:
1. Какую 1 ключевую функцию можно выделить?
2. Какие альтернативные названия подойдут?
3. Какой сегмент аудитории недообслужен?
```

### Для оценки конкурента:
```
Проанализируй расширение [ССЫЛКА]:
- Основной функционал
- Модель монетизации
- Слабые места (по отзывам)
- Возможности для улучшения
```

## Типичные ошибки

1. **Слишком сложная идея** — расширение должно решать 1 проблему
2. **Игнорирование конкурентов** — нужно проверить занятость ниши
3. **Мало идей** — нужно минимум 5-10 гипотез для сравнения
4. **Нет проверки монетизации** — идея должна иметь потенциал заработка
