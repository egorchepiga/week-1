# Этап 7: Проверка занятости запроса конкурентами

## Цель этапа
Убедиться, что выбранный ключ не занят оптимизированными расширениями-конкурентами.

## Почему это критически важно

```
ФАКТ: Основной трафик приходит из Google, НЕ из поиска CWS!

ПУТЬ ПОЛЬЗОВАТЕЛЯ:
  1. Пользователь вводит запрос в google.com
  2. Google показывает расширения в выдаче
  3. Пользователь переходит на страницу расширения в CWS
  4. Устанавливает расширение

ВЫВОД: Нужно конкурировать в Google, а не в CWS!
```

## Три метода проверки конкурентов

### Метод 1: Google Search
```
ДЕЙСТВИЕ: Открыть google.com (желательно US VPN)
ЗАПРОС: [ваш ключ] chrome

ПРИМЕР: "site blocker chrome"

ЧТО СМОТРЕТЬ:
  ✓ Есть ли расширения в топ-10?
  ✓ Какие названия у конкурентов?
  ✓ Совпадает ли их название с запросом?

ХОРОШИЙ ЗНАК: Нет расширений или они не оптимизированы
ПЛОХОЙ ЗНАК: Топ-5 заняты расширениями с точным match названия
```

### Метод 2: Semrush SERP Analysis
```
ДЕЙСТВИЕ: Keyword Overview → прокрутить вниз до SERP

ЧТО СМОТРЕТЬ:
  ✓ Как выглядит американский фид
  ✓ Какие типы результатов (сайты, расширения, видео)
  ✓ Позиции расширений в выдаче

ПРЕИМУЩЕСТВО: Видим выдачу глазами американцев
```

### Метод 3: app-database.com (ЛУЧШИЙ)
```
ДЕЙСТВИЕ: Вбить название в поиске на app-database.com

ЧТО ПРОВЕРЯТЬ У КОНКУРЕНТОВ:
  ✓ Название включает те же слова, что и ваше
  ✓ Описание более 3,000 символов
  ✓ Более 30 переводов

ПРИМЕР НЕЗАНЯТОГО:
  Ищем "focus mode block" → нет результатов → СВОБОДНО!

ПРИМЕР ЗАНЯТОГО:
  Ищем "site blocker" → есть конкурент с 3K+ описанием и 50 переводами → ЗАНЯТО
```

## Критерии оптимизированного конкурента

```
КОНКУРЕНТ ОПТИМИЗИРОВАН, ЕСЛИ:
  ✗ Название содержит ваш ключ
  ✗ Описание >3,000 символов
  ✗ >30 переводов
  ✗ Регулярные обновления
  ✗ Много отзывов (>100)

КОНКУРЕНТ НЕ ОПТИМИЗИРОВАН, ЕСЛИ:
  ✓ Название не совпадает с запросом
  ✓ Короткое описание (<1000 символов)
  ✓ Мало переводов (<10)
  ✓ Давно не обновлялся
  ✓ Мало отзывов
```

## Стратегии при разной конкуренции

### Запрос свободен
```
СИТУАЦИЯ: Нет расширений с таким названием

ДЕЙСТВИЯ:
  ✓ Срочно запускаться!
  ✓ Базовой оптимизации достаточно
  ✓ Занять нишу первым

ПРИМЕР: "bing chatgpt" — был свободен → первый занял топ
```

### Есть неоптимизированные конкуренты
```
СИТУАЦИЯ: Конкуренты есть, но не оптимизированы

ДЕЙСТВИЯ:
  ✓ Сделать полную оптимизацию
  ✓ Описание 3000+ символов
  ✓ 50+ переводов
  ✓ Качественные скриншоты
  ✓ Регулярные обновления

ШАНСЫ: Высокие — обгоните за 1-2 месяца
```

### Есть оптимизированные конкуренты
```
СИТУАЦИЯ: Конкуренты есть и оптимизированы

СТРАТЕГИИ:

A) Нишевание:
   "grammar check" занят → "grammar check for Gmail"

B) Альтернативная формулировка:
   "site blocker" занят → "focus mode"

C) Добавление слова:
   "website blocker" занят → "web site blocker"
   (ВАЖНО: проверить что новый вариант свободен!)
```

### Запрос полностью занят топами
```
СИТУАЦИЯ: Топ-5 занят крупными оптимизированными расширениями

ДЕЙСТВИЯ:
  ✗ НЕ брать этот запрос
  ✓ Искать альтернативный ключ
  ✓ Использовать методы из уроков 4-5

ПРИМЕР: "adblocker" — занят AdBlock, uBlock и др.
```

## Допустимые и недопустимые совпадения

### Допустимо:
```
✓ Отдельные слова совпадают с конкурентами

ПРИМЕР:
  Конкурент: "site blocker"
  Ваше название: "web site blocker" — OK (если свободно)

ПРИМЕР:
  Конкурент: "site blocker"
  Ваше название: "focus mode" — OK (совсем другие слова)
```

### Недопустимо:
```
✗ Полное совпадение с оптимизированным конкурентом

ПРИМЕР:
  Конкурент: "Site Blocker Pro" (оптимизирован)
  Ваше название: "Site Blocker" — ПЛОХО

✗ Только добавление Pro/Plus/Free к занятому названию
```

## Преимущество расширений над сайтами

```
ФАКТ: Google любит расширения и продвигает их!

ПРИМЕР:
  Запрос: "dark theme"
  Результат: Расширение в топ-3, хотя сайтов тысячи

ПРИМЕР:
  Запрос: "email checker"
  Результат: Расширения выше огромных email-сервисов

ВЫВОД: Даже при высокой конкуренции от САЙТОВ,
       расширение может пробиться в топ
```

## Подсказки для промптов

### Для анализа конкурента:
```
Проанализируй расширение-конкурента "[НАЗВАНИЕ]":
1. Насколько оптимизировано описание?
2. Сколько переводов?
3. Частота обновлений?
4. Качество отзывов?
5. Слабые места?
```

### Для поиска свободной ниши:
```
Для ниши "[НИША]" найди незанятые варианты названий:
- Основные конкуренты: [список]
- Их ключевые слова: [список]

Предложи 10 альтернативных формулировок, которые:
- Не совпадают с конкурентами
- Имеют поисковый потенциал
- Понятны пользователю
```

### Для оценки шансов:
```
Оцени шансы на успех для расширения "[НАЗВАНИЕ]" в нише "[НИША]":

Конкуренты:
1. [Конкурент 1] - [пользователи] - [оптимизация: да/нет]
2. [Конкурент 2] - [пользователи] - [оптимизация: да/нет]

Какова вероятность попасть в топ-5 за 3 месяца?
```
