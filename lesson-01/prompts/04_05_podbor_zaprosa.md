# Этапы 4-5: Подбор запроса для названия продукта

## Цель этапа
Найти ключевое слово для названия расширения, которое обеспечит 70% успеха в SEO.

## Важность названия
```
ПРАВИЛО: Ключевое слово в названии = 70% успеха SEO

ПОЧЕМУ:
- Google ранжирует по совпадению с названием
- Пользователи ищут конкретные фразы
- Расширения с точным match получают приоритет
```

## Алгоритм действий

### Шаг 4.1: Настройка Semrush
```
ДЕЙСТВИЕ: Зарегистрироваться на semrush.com
ВАЖНО: Использовать антидетект-браузер (даже за рубежом!)

ВАРИАНТЫ ДОСТУПА:
  1. Pro trial (7 дней бесплатно, можно отменить)
  2. plati.market/itm/3217737 — доступ за 450₽ на 14 дней

НУЖНЫЕ ИНСТРУМЕНТЫ:
  - Keyword Overview
  - Keyword Magic Tool
```

### Шаг 4.2: Анализ ключевого слова в Keyword Overview
```
ДЕЙСТВИЕ: Ввести ключевое слово конкурента, нажать Search

МЕТРИКИ НА ДАШБОРДЕ (пример: "site blocker"):
┌─────────────────────────────────────────────────────────┐
│ 2.4K  — запросы в US БЕЗ вариаций (хвостов)            │
│ 42.6K — запросы в МИРЕ БЕЗ вариаций                    │
│ 103.5K — запросы в US С вариациями (long tails)        │
└─────────────────────────────────────────────────────────┘

РАСЧЁТ МИРОВОГО ТРАФИКА С ХВОСТАМИ:
  Пропорция US/World = 2.4K / 42.6K = 5.6%
  Мировой трафик = 103.5K / 5.6% × 100% ≈ 1.85M запросов
```

### Шаг 4.3: Минимальные пороги трафика
```
ШИРОКИЕ НИШИ (site blocker, adblocker, focus mode):
  ✓ Минимум 2,000 запросов/мес в US (без хвостов)

УЗКИЕ РАБОЧИЕ НИШИ (color picker, chatgpt for linkedin):
  ✓ Минимум 500 запросов/мес в US (без хвостов)

ВАЖНО: Это запросы БЕЗ вариаций!
```

### Шаг 4.4: Поиск альтернативных ключей

#### Метод 1: Keyword Magic Tool — популярные слова в хвостах
```
ДЕЙСТВИЕ:
  1. Включить "By volume"
  2. Смотреть столбик слева с популярными словами
  3. Комбинировать слова для нового названия

ПРИМЕР: "site blocker" → видим популярные слова:
  website, block, chrome, free, best
  → Новое название: "Website Block Pro"
```

#### Метод 2: Keyword Magic Tool — Related запросы
```
ДЕЙСТВИЕ:
  1. Включить "Related"
  2. Включить "By volume"
  3. Найти жирные похожие запросы

ПРИМЕР: "chatgpt" → Related → "bing chatgpt" (незанятый!)
```

#### Метод 3: Google.com
```
ДЕЙСТВИЕ: Искать идеи в выдаче Google
  - Смотреть, какие запросы не заняты расширениями
  - Анализировать сайты и приложения в выдаче
```

#### Метод 4: ChatGPT для синонимов
```
ПРОМПТ:
  Find 10 synonyms to the following keywords: [ваш запрос]

ПРИМЕР:
  Find 10 synonyms to the following keywords: site blocker, block website

ИСПОЛЬЗОВАНИЕ: Берём кусочки из ответов для поиска новых ключей
```

### Шаг 4.5: Критерии идеального ключа

```
ИДЕАЛЬНЫЙ КЛЮЧ ДОЛЖЕН:
  ✓ Иметь достаточный трафик (см. пороги выше)
  ✓ Не быть занятым оптимизированными конкурентами
  ✓ Быть "софтовым" (ассоциироваться с программами)
  ✓ Быть понятным пользователю
  ✓ Отличаться от названий конкурентов
```

## Стратегии для разных ситуаций

### Жирный запрос занят
```
СТРАТЕГИЯ: Нишевание

ПРИМЕР:
  "grammar check" занят → "grammar check for Gmail"
  "site blocker" занят → "focus mode site blocker"
```

### Нужно больше трафика
```
СТРАТЕГИЯ: Поиск супер-популярного слова

ПРИМЕР: Вбить "chatgpt" в Keyword Magic Tool
  → Найти популярную незанятую пару
  → "bing chatgpt" — 50K+ запросов, низкая конкуренция
```

### Конкуренты похожи
```
СТРАТЕГИЯ: Альтернативная формулировка

ПРИМЕР:
  Конкуренты: "site blocker", "website blocker"
  Альтернатива: "focus mode", "distraction blocker"
```

## Чек-лист завершения этапа

- [ ] Semrush настроен и работает
- [ ] Проанализирован ключ конкурента
- [ ] Проверено минимум 5-10 альтернативных ключей
- [ ] Найден ключ с достаточным трафиком
- [ ] Ключ не занят оптимизированными конкурентами
- [ ] Записаны метрики выбранного ключа

### Для генерации вариантов названия:
