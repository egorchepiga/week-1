## ЭТАП 6: KEYWORD DIFFICULTY

### Правило:
```
KD рассчитан для сайтов, НЕ для расширений!
Расширения могут пробиваться даже при KD=90%
```

### Мультипликаторы трафика:
```
US → Мир (английский) = ×10
Английский → Все языки = ×10
500 US запросов ≈ 50,000 мировых
```

## ЧАСТЫЕ ОШИБКИ И РЕШЕНИЯ

| Ошибка | Решение |
|--------|---------|
| Слишком много функций | Оставить 1 главную |
| Короткое описание | Расширить до 3000+ символов |
| Мало переводов | Добавить 30-50 языков |
| Занятый ключ | Нишевать или искать альтернативу |
| Manifest V2 | Мигрировать на V3 |
| Нет Privacy Policy | Создать на бесплатном генераторе |
