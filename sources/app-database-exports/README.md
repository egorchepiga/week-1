# Экспорты из app-database.com

## Источник
- **URL:** https://app-database.com
- **Дата экспорта:** 2025-12-04
- **Тип аккаунта:** Premium (через t.me/the_bootcamp_bot)

## Как получили
1. Зарегистрировались через Telegram бота
2. Активировали аккаунт на app-database.com/profile
3. Использовали фильтры для поиска расширений
4. Экспортировали результаты в XLSX

## Файлы

20 XLSX файлов с различными категориями расширений:
- Топ по пользователям
- AI/ChatGPT расширения
- Developer tools
- Productivity
- Social Media
- и другие категории

## Структура XLSX

| Колонка | Описание |
|---------|----------|
| Name | Название расширения |
| Users | Количество пользователей |
| Rating | Рейтинг (1-5 звёзд) |
| Languages | Количество переводов |
| Category | Категория в CWS |
| Last Updated | Дата обновления |
| Description | Описание |

## Как обрабатывать

```bash
# Установить зависимости
pip install pandas openpyxl

# Запустить анализ
cd ../../../scripts
python analyze_exports.py

# Результат: reports/ANALYSIS_SUMMARY.md
```
