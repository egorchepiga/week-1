# Транскрипция вебинара Captain Builders

## Источник
- **Файл:** videoplayback.mp3 (185 MB)
- **Продолжительность:** ~2 часа
- **Дата транскрипции:** 2025-12-04
- **Модель Whisper:** turbo

## Как получили
1. Скачали аудио вебинара буткемпа
2. Транскрибировали через OpenAI Whisper

## Как транскрибировать

```bash
# Установка (один раз)
cd C:\Users\George\Desktop\startup\whisper
.\transcribe.bat

# Или вручную:
pip install -U openai-whisper
whisper audio.mp3 --model turbo --language Russian --output_format txt
```

## Ключевые инсайты из вебинара

### Нишевые vs Широкие проекты

| Критерий | Нишевый | Широкий |
|----------|---------|---------|
| Пользователей | от 10K | от 100K |
| Запросов US/мес | от 500 | от 2000 |
| Выручка на юзера | Высокая | Низкая |
| Конкуренция | Ниже | Выше |

### Признаки нишевого проекта:
1. Ориентирован на работу (8-10 баллов)
2. Тематичный (одна профессия)

### Формулы из вебинара:

```python
# Примерная месячная выручка
monthly_revenue = (users / 100) * subscription_price

# Стоимость продажи проекта (без выручки)
sale_price = users / 10

# Мировой трафик из US
global_traffic = us_traffic * 10  # англоязычный
all_languages = global_traffic * 10  # все языки
```

### Критерии оптимизированного конкурента:
1. Название = точный поисковый запрос
2. Description > 3000 символов
3. Языков > 30

### Если хотя бы 1 критерий не выполнен — можно занимать эту нишу!
