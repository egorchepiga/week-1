## Ключевые концепции

### Источники готового кода
1. **Open Source** (GitHub) — проверить лицензию!
2. **LLM** (ChatGPT, Claude) — генерация с нуля
3. **Большие проекты** — выдернуть одну функцию

### Критерии простого проекта
- Одна функция
- Реализация за 2-5 дней
- Можно взять из open source
- Можно сгенерировать через LLM

### Бюджет на разработку
- ~$300 если заказываешь
- $0 если партнёр-разработчик
- $0 если сам через LLM
