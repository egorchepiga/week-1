```
Test and evaluate XPath expressions on any webpage.
Best xpath helper, finder and checker tool for developers.
```

### Long Description (3000+ символов)

### Features для описания
- Test XPath expressions on any webpage
- Real-time XPath evaluation
- Highlight matching elements

- Support for both XPath 1.0 and CSS selectors
