## Ключевые выводы

### 1. Ниша IMPORT полностью свободна
- В Chrome Web Store **НЕТ НИ ОДНОГО** расширения для импорта JSON → Google Sheets
- Все существующие расширения делают EXPORT (Sheets → JSON)
- Это уникальная возможность занять пустую нишу

### 2. Софтовая выдача Google
- Топ-10 Google для IMPORT запросов: туториалы, Q&A, видео
- Нет оптимизированных Chrome Extensions
- Идеальные условия для SEO-продвижения

### 3. Рекомендуемая стратегия
- **Основной ключ:** `json to google sheets` (210 Volume, 28% KD)
- **Функция MVP:** IMPORT (JSON → Sheets)
- **Дополнительно:** EXPORT как secondary feature
