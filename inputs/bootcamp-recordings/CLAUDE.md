## Ключевые инсайты (быстрая справка)

### Формула трафика
```python
real_traffic = us_volume × 10 × 10  # ×10 EN global, ×10 все языки
```

### Критерии оптимизации
- Название = Запрос (точно)
- 3000+ символов Long Description
- 30+ языков

### Чек-лист валидации
```
□ Volume >= 500?
□ KD <= 70?
□ SERP софтовый?
□ Есть платные конкуренты?
□ Нет MV2 зависимости?
□ Для работы (не развлечение)?
```

## Промпты использованные для обработки

### Извлечение ошибок
```
Прочитай транскрипцию. Извлеки ВСЕ упомянутые ошибки.
Для каждой: название, описание, почему плохо, решение.
```

### Создание выжимки
```
Читай по частям. Создай:
1. Главные инсайты
2. Разобранные идеи + вердикты
3. Ключевые правила
4. Лучшие ниши
5. Флоу валидации
```

