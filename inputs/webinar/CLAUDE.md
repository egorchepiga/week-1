## Ключевые темы (из WEBINAR_INSIGHTS.md)

### 1. Формулы расчёта
```python
monthly_revenue = (users / 100) * price   # 1% конверсия
sale_price = users / 10                    # продажа проекта
global_traffic = us_traffic * 10 * 10     # ×10 EN, ×10 все языки
```

### 2. Критерии нишевости
- Связь с работой/профессией
- Одна аудитория
- Минимум 10,000 пользователей (нишевый) / 100,000 (широкий)
- Минимум 500 запросов US/мес

### 3. Признаки оптимизированного конкурента
| Критерий | Значение |
|----------|----------|
| Название | = точный поисковый запрос |
| Description | > 3,000 символов |
| Локализация | > 30 языков |

### 4. Запрещённые бренды
**НЕТ:** Meta, LinkedIn, Claude, Gemini
**ДА:** YouTube, Amazon, Google Sheets

### 5. KD и софтовость
- KD до 30% - можно брать
- Софтовость: есть ли расширения в топе выдачи
